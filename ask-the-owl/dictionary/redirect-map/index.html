Redirect map

Redirect map

A redirect map is a plan that shows where old URLs should send visitors and search engines after a website move or page change.

Plain-English version

It is the relocation chart that keeps visitors from getting lost when the address list changes.

Why it matters

Without a redirect map, migrations shed traffic, confuse users, and create unnecessary 404s.

Where it shows up

Platform moves, service-page restructures, mergers, rebrands, and clean URL rebuilds all need redirect logic.
