# thepresidentofai.github.io

## Purpose

GitHub Pages site for `Boho Digital Services`, hosted from the `thepresidentofai` account.

## Current Contents

- `index.html` - main marketing homepage.
- `ask-the-owl/` - Ask the Owl hub plus the Owl Dictionary index and entry pages.
- `bad-seo-field-guide/` - Shortcut Fox warning terms and glossary-linked red flags.
- `contact/index.html` - project inquiry and contact brief page.
- `services/provider-transfer-support/` - provider transfer support page with glossary-linked migration language.
- `work/` and `our-impact/` - new brand-aligned top-level routes.
- `404.html` - custom not-found page.
- `assets/css/styles.css` - shared site styles.
- `assets/js/site.js` - navigation, reveal, copy-prompt, and glossary behavior.
- `assets/js/owl-dictionary-data.js` - centralized Owl Dictionary term data and short definitions.
- `content/dictionary/README.md` - content governance for dictionary terms and inline highlights.
- `assets/img/` - hero, process, and favicon assets.
- `robots.txt` - crawl policy plus sitemap reference.
- `sitemap.xml` - current public page index.
- `LICENSE` - MIT license created with the GitHub repository.

## Setup Notes

The SSH key should live outside this repo under the local Windows SSH directory. Add only the public key to GitHub.

Remote: `git@github.com:thepresidentofai/thepresidentofai.github.io.git`

## Dictionary Governance

The Owl Dictionary is maintained as a content product. Short definitions live in `assets/js/owl-dictionary-data.js`, and editing rules live in `content/dictionary/README.md`.
