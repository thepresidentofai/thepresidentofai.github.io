Owl Definition

Indexing

Indexing is the process search engines use to store and organize pages so they can appear in search results.

Plain English

A page can exist and still fail to show up in search if it never makes it into the search engine's index or if it gets dropped later.

Common blockers

Weak crawl paths, accidental noindex settings, duplicate pages, poor redirects, or messy migrations can all create indexing trouble.
